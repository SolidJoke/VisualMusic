Guide d'utilisation

swipe_up

Interface Tactile

La barre de navigation inférieure permet un accès rapide aux sections principales (Studio, Dictionnaire, Lecture). Balayez vers le haut pour ouvrir les paramètres avancés de chaque module.

view_agenda

Mode Studio

Sur mobile, les instruments sont empilés verticalement. Tapez sur l'en-tête d'un instrument pour le développer et accéder aux contrôles de fretboard ou de clavier en plein écran.

bluetooth_connected

Connectivité

Connectez des contrôleurs MIDI via Bluetooth LE ou USB. Le statut de connexion est indiqué par le point vert dans la barre supérieure de l'espace de travail.
