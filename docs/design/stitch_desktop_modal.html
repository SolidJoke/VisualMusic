Guide d'utilisation

piano

Configuration MIDI

Connectez vos instruments via USB ou Bluetooth MIDI. L'application détectera automatiquement vos appareils et configurera le routage de base pour une latence minimale.

monitoring

Analyse de Performance

Visualisez votre précision rythmique et votre dynamique en temps réel. Le panneau de progression (P2) s'affiche au bas de l'écran pour un retour non intrusif.

layers

Multi-fenêtrage

Détachez les panneaux secondaires (Effets, Bibliothèque) pour les placer sur un second moniteur. Utilisez l'icône "Détacher" dans l'en-tête de chaque panneau.

keyboard_command_key

Raccourcis Clavier

Accélérez votre flux de travail en maîtrisant les commandes rapides. Appuyez sur Ctrl + / pour afficher l'aide-mémoire à tout moment.
