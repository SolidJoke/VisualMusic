## 2. Instrument Specific Logic

### Piano: Inversion & Fingering
*   **Root**: 1-3-5
*   **1st Inversion (Bass on 3rd)**: 1-2-5 (Recommended to avoid tension)
*   **2nd Inversion (Bass on 5th)**: 1-3-5
*   **Extensions (9, 11, 13)**: Omit the 5th if necessary for playability.

### Guitar: CAGED System
*   **C Shape**: Root on String 5
*   **A Shape**: Root on String 5 (Common barre)
*   **G Shape**: Root on String 6 (Difficult barre, good for arpeggios)
*   **E Shape**: Root on String 6 (Most common barre)
*   **D Shape**: Root on String 4 (High register)

### Bass: The "Clarity" Rule
*   **Avoid Muddiness**: No full triads in low register.
*   **Dyads (1-5)**: Rock/Metal stability.
*   **Tenths (1-10)**: Expert recommendation for clarity. (e.g., G on E-string, B on G-string).
