# VisualMusic Genre Ontology (V1.0)

Derived from SAS Analysis (Doc 1). This document defines the technical parameters for the composition and visualization engine.

## 1. Jazz & Bossa Nova (Sophisticated Harmony)

| Parameter | Specification |
| :--- | :--- |
| **Harmonic Density** | High (7th, 9th, 11th, 13th) |
| **Bass Behavior** | Walking Bass (Quarter notes) or Syncopated (Bossa) |
| **Rhythm** | Swing feel (Triplet shuffle) |
| **Voicings** | Shell Voicings (1-3-7) to avoid muddiness |

## 2. Rock & Metal (Power & Energy)

| Parameter | Specification |
| :--- | :--- |
| **Harmonic Density** | Low (Triads, Power Chords 1-5) |
| **Bass Behavior** | Root-oriented, driving eighth notes |
| **Rhythm** | Straight 4/4, heavy accents on 1 and 3 |
| **Voicings** | E-Shape and A-Shape barres |

## 3. Reggae & Dub (The "Offbeat")

| Parameter | Specification |
| :--- | :--- |
| **Harmonic Density** | Moderate (Triads or 7ths) |
| **Bass Behavior** | Melodic, sparse, focus on low frequencies |
| **Rhythm** | "Skank" on beats 2 and 4 |
| **Voicings** | Short, percussive (Staccato) |

## 4. Pop & Funk (Clarity & Groove)

| Parameter | Specification |
| :--- | :--- |
| **Harmonic Density** | Variable (Focus on "Add9" and "Sus" colors) |
| **Bass Behavior** | Syncopated, "Slap" potential |
| **Rhythm** | 16th note syncopation |
| **Voicings** | "The Island Strum" pattern |

## 5. Urban (Hip-Hop, Trap, R&B)

| Parameter | Specification |
| :--- | :--- |
| **Harmonic Density** | Lush (Minor 9ths, Neo-Soul extensions) |
| **Bass Behavior** | "808" style (Sub-bass, long sustain) |
| **Rhythm** | Laid back (Behind the beat) |
| **Voicings** | Tight clusters in the mid-register |
